# Number Guessing Game (GUI Version)

## Description
This is a simple Python GUI project using "Tkinter".
The computer randomly selects a number between 1 and 100.
The player has to guess the number, and the program gives feedback:
- "Too low"
- "Too high"
- "Correct!"

## How to Run
1. Make sure Python 3 is installed.
2. Open `number_guess_gui.py` and run it.
3. Enter your guess in the GUI and click "Check".
